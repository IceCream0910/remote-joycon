## 스마트폰으로 PC 게임 컨트롤러 구현하기
Developing a PC Game Controller with a Smartphone

### Used Stacks
- 통신 : WebSocket
- 조이스틱 : [nippleJS](https://yoannmoi.net/nipplejs/)
- 키입력 : [pydirectinput](https://pypi.org/project/PyDirectInput/)

### Required Modules
- asyncio
- threading
- websockets
- pydirectinput
- time
